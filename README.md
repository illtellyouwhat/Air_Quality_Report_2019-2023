# Air_Quality_Report_2019-2023
 Worldwide report on pollution from 2019-2023 :  Is air pollution getting better?  What do we need to pay attention to?

 This repo is designed in the following way:
 
 Documentation.md is the full story of how this analysis came together. It has a highlevel overview called TLDR briefly explaining the method and steps used.  Following that is a detailed description of the process step by step in its execution. IF so desired with a basic knowledge of Python and SQL within their respective IDEs the user should be able to recreate this entire analysis step by step.
